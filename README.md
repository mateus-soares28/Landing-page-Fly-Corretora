# Landing Page Fly Corretora

Landing page institucional para a Fly Corretora de Seguros, com foco em apresentação dos serviços, chamadas para cotação via WhatsApp, depoimentos e formulário de contato.

## Tecnologias

- HTML5
- CSS3
- JavaScript
- GSAP + ScrollTrigger
- Font Awesome
- Google Fonts

## Estrutura

```text
.
├── index.html
├── css/
│   ├── reset.css
│   ├── style.css
│   └── responsive.css
├── js/
│   └── script.js
└── img/
```

## Como executar

Abra o arquivo `index.html` diretamente no navegador.

Não é necessário instalar dependências ou rodar servidor local.

## Principais recursos

- Layout responsivo para celular, tablet, notebook e desktop.
- Animações de entrada com GSAP.
- Cards de seguros.
- Depoimentos em formato de pilha interativa.
- Formulário que envia as informações preenchidas para o WhatsApp da empresa.
- Links diretos para cotação via WhatsApp.

## Arquivos principais

- `index.html`: estrutura da página.
- `css/style.css`: estilos principais.
- `css/responsive.css`: ajustes responsivos.
- `js/script.js`: animações, interações, formulário e depoimentos.
